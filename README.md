# Weather App

A Flutter weather application that fetches real-time weather data from an external API.  
The app automatically detects the userâ€™s current location to display local weather information and allows searching for weather details in other cities.

## Features

- Detects user location and shows current city weather
- Fetches real-time weather data from an API
- Search weather by city name
- State management using **Cubit (flutter_bloc)**
- Clean and responsive UI

## Technologies Used

- **Flutter**
- **Dart**
- **Cubit (flutter_bloc)**
- **Dio package**
- **Location / Geocoding**

## State Management

This project uses **Cubit** for state management to:
- Handle API calls
- Manage loading, success, and error states
- Keep the UI clean and reactive
